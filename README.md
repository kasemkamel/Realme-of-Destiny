# Battle Client

## Installation

### Requirements

- Python 3.8+
- Pygame 2.0.1+
- aiohttp 3.8.1+
- aioredis 2.0.1+
- numpy 1.21.2+
- pydantic 1.9.0+
- fastapi 0.68.1+
- uvicorn 0.15.0+

### Installation

```bash
pip install -r requirements.txt
```

## Usage

### Run

```bash
python battle_client.py
```
